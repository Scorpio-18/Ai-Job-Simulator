🎯 AI Job Interview Simulator

The AI Job Interview Simulator is an interactive web-based tool designed to help students and job seekers practice real interview scenarios. Built entirely using HTML, CSS, and JavaScript, this simulator uses Google Web Speech API and AI APIs to generate personalized interview questions and give structured feedback.

This project was created by me and my friend during our 6th semester as part of our major project.

🚀 Features

AI-Generated Interview Questions based on user bio & job role

Voice Input Support using Google Web Speech API

Real-time Feedback (AI-based or rule-based depending on version)

Light/Dark Mode UI

Progress Tracking

Shareable Feedback Summary

Modern & Clean Frontend built with HTML, CSS, and JS

Google Sheets Integration (optional) for saving reports

🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Speech Recognition: Google Web Speech API

AI Models: OpenAI / Gemini / HuggingFace (free-tier friendly)

Storage: Google Sheets / Local Storage

Version Control: Git & GitHub

📌 How It Works

User enters their bio + role (e.g., Frontend Developer, Data Analyst).

System generates 10 personalized interview questions.

Candidate answers using mic or typing.

At the end, system generates feedback with suggestions & improvements.

Feedback can be saved or shared.

✨ Project Status 

Completed and functional. Future improvements may include enhanced AI feedback and backend integration.
